# accounting-firm
A responsive business website developed for a local accounting firm to enhance its online presence and client accessibility. Built using HTML, CSS, and JavaScript, with focus on clean UI, structured layout, and performance optimization. Deployed via GitHub Pages as part of my transition into Computer Science and software development.

🧠 Project Motivation
This project was developed to support a real-world accounting firm by providing a clean, structured, and professional online presence. It also reflects my transition into Computer Science and software development, demonstrating applied knowledge in web technologies and deployment workflows.

🛠️ Technologies Used
	•	HTML5 — website structure and semantic layout
	•	CSS3 — styling, layout, and responsive design
	•	JavaScript — interactive behavior and dynamic elements
	•	GitHub Pages — static website deployment

📦 Features
	•	Professional landing page introducing the firm
	•	Services overview section
	•	Structured navigation
	•	Responsive layout for different screen sizes
	•	Clean, user-focused interface
